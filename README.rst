===============================
shopidown
===============================


Simple markdown parser for Shopify


* Free software: MIT license


Features
--------

* Parses simple markdown files

How to run the tests
--------------------
::

    git clone https://github.com/douglas/shopidown.git
    cd shopidown
    python setup.py test
