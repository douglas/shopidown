===============================
shopidown
===============================


Simple markdown parser for Shopify


* Free software: MIT license
* Documentation: https://shopidown.readthedocs.io.


Features
--------

* Parses simple markdown files

```bash
https://github.com/douglas/shopidown.git
cd shopidown
python setup.py test
```
